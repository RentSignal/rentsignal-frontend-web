# rentsignal-frontend-web

렌트시그널 서비스의 프론트엔드 웹 애플리케이션 레포지토리

## install

1. npm 설치

```
npm install
```

2. .env 파일 생성

```
VITE_KAKAO_JS_KEY=JAVASCRIPT_KEY 여기에
```

- [프론트엔드 환경변수 페이지 🔗](https://www.notion.so/env-306064094cb0808ba58dd06c45442abb)

3. 실행

```
npm run dev
```
